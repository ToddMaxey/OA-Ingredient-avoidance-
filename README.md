### **Ingredients to Avoid**

**Key Words:**

- **Flour**
- **Fruit Juice Concentrate**
- **Honey**
- **Malt**
- **Meal**
- **Molasses**
- **Nectar**
- **Starch**
- **Sugar**
- **Sweetener**
- **Syrup**

**Suffixes Indicating Sugars and Sugar Alcohols:**

- **-ose**  
  *(e.g., glucose, fructose, sucrose, dextrose, maltose, lactose)*

- **-itol**  
  *(e.g., sorbitol, xylitol, mannitol, erythritol)*
